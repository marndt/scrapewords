# scrapewords

As described in FB-3275, scrapewords will make a web request for each of the URLs in the input file
and then add the words found in each of the pages to an output file in JSON format.

## Installation
```pip install git+https://github.com/marndt/scrapewords```

## Usage
```scrapewords url_list.txt output.json```

## Example input
```
http://google.com/
http://nurse.org/
```
